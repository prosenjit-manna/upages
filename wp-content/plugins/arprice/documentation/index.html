Template Selection

By clicking on "ARPrice" link in wordpress left menu, you will get list of all the pricing tables. Select your favorite template from list of templates. you can see all thumbnails. You can either edit existing template or create clone of existing template. I would suggest to clone it.

1) Edit Template

By Editing template, all the changes will be saved in original template.

  • Click on Edit Button of any Pricing Table which you want to Edit.

  • Double Click on any Pricing Table, which you want to Edit.

  • Select any Pricing Table and click on Select/Choose Button.

    • 2) Clone Template

    This functionality will create a new clone template of existing template and all the changes will be saved into clone template. Your original template will stay as is.

  • Click on Clone button position at particular pricing table to create Clone of that template.

  • Select any Pricing Table to create Clone of that and Click on Clone button positioned at right upper corner.

    • Real Time Editor

    ARPrice provides excellent facility for edit template at design time. From Editor you can do any changes related to template and all changes you have set will be applied to your template right away. ARPrice is the only pricing table plugin worldwide to provide this type of state of the art Editor. Its real user friend that any novice person too can create beautiful pricing tables and put on page without having any programming knowledge. In below picture it shows you basic view of editor. Detailed explanation of each portion is explained below.

    Color Options

    In above picture you can see that on top right corner there is facility to change color of your pricing table as per your requirement. Just click your favorite color and it will be applied to your pricing table Right away.

    Column Settings

    ARPrice provides effective functionality to manage column in an easy way. You can rearrange columns by Drag & Drop according to your need.

    1) Drag and Drop Column (Rearrange Column)

    To sorting column in package just simply drag and drop at position.

    2) Resize Column

    ARPrice provides facility to resize column as per template requirement. You just have to select column and resize the column.

    3) Column General Options

    From here you can set general settings for column. you can set column width, and also able to hide column or highlight column,etc.

    4) Ribbon Options

    You can add ribbon to package column from here. ARPrice pricing table provides plenty of options for ribbon. Ribbons are totally based on CSS. No images used. You can set ribbon text, ribbon background color, ribbon text color, it's position and much more.

    5) Add New Column

    As per its name suggest, you can add new column easily from here by just click on Add New Column

    6) Duplicate Column

    If you want to duplicate any column in template, simply select column and click on duplicate button.

    7) Delete Column

    If you want to delete any column from template, simply select column and click on delete button.

    Header Settings

    ARPrice pricing table plugin allows you to set Header settings like Column Title and Column Description and its styling. To select Header section, "Double Click" on it. From here you can change your Column Title content, font styling , font-size, color,etc. Even you can set image or any icon and also supports video in header.

    Price Settings

    ARPrice allows you to set various options related to Price text. From here you can set font style, color, size for price text and even you can add Font Awesome Icon also. You can manage Pricing and its interval from this area.

    Body Settings

    Here you will be able to manage all body options like font style, font size, font family, font color and text alignment etc. "Double Click" on Body Section for Edit.

    Row Settings

    ARPrice allows you to change particular cell content, add tooltip on it, add font awesome icon,etc. To select any row "double click" on body section and then "double click" on row. From here you can duplicate any row and also you can create new row. To delete any row, select row and click 'Delete Row' button.

    1) Drag and Drop Row (Rearrange Row)

    To arrange row in package, simply drag and drop at position as per your need.

    2) Add New Row

    ARPrice allows you to add new row from here also by just one click. Select any row and click on Add New Row icon.

    3) Duplicate Row

    ARPrice also allows you to duplicate column as well as row. Yes! here you have to select Row and click on Duplicate icon.

    4) Delete Row

    From here you can delete particular row, just simply select the row and click on Delete icon.

    Button Settings

    From here you can set button settings of particular package. To select button, "Double Click" on button. You can set displayed text on button, text styling, set action performed after clicking on button. You can set image on button and also Font Awesome Icon. So from here you can customize button as per your requirement in simple way.
    You can add custom button script like "PayPal" or other shortcodes to perform various action after button click.

    1) Media Supports

    If you want to set background image instead of button text, you can do that from here.

    2) Font Awesome Icon

    If you want to add icon with button text, you can do that from here. You can select any size of icon given from list.

    3) Button Size

    ARPrice allows you to change button size as per your template requirement. You can set Large, Medium or Small size button.

    4) Button Link

    ARPrice allows you to manage redirection after clicking on button and also you will be able to open link in current window or in new window.

    5) Button Script

    ARPrice also facilitates you to add any script on button(e.g. PayPal Code).

    General Options

    ARPrice provides various template level options to manage pricing table columns. All changes will affect whole pricing table.

    1) Column Options

    You can set whether template will be responsive or not. You can also set column width or space between two columns etc from this section.

    ARPrice pricing table also provides facility to set Opacity of your pricing table. If you have use any image in your pricing table and you want to give opacity, you can do that from here by just select value for opacity and "save" pricing table.

    2) Effects

    From “Effects” option you can set animation on column. You can set how many columns will be displayed at a time, set how many columns to be rotate, pagination settings of column, scroll effect, etc.

    3) Tooltip

    You can apply different styling to tooltip text and set background color and also can set font awesome icon as per template look.

    4) Custom CSS

    To Customize Template as per your requirement, you can set css for any class from here. If you want add your css to pricing table or overwrite any default css then you can do that from here in simple way. Just pick up any class of pricing table where you want do changes and put it in Custom Css block and save pricing table.

    e.g.
    .arp_price_wrapper {
     color : #990000;
     font-size : 15px;
    }

    Pricing Table Preview

    ARPrice pricing table provides realtime preview of pricing table. It also provides facility to view template in responsive layout. After done necessary changes, you can view your template runtime either of 3 ways:

    1) Desktop View

    You can preview desktop view by clicking on "Desktop" icon.

    2) Tablet View

    You can preview table view by clicking on "Tablet" icon.

    3) Mobile View

    You can preview mobile view by clicking on "Mobile" icon.

    Publish Your Pricing Table

    ARPrice facilitates you to display pricing table into your page or post by just shortcode. You just need to place shortcode retrieved from ARPrice plugin, where you want to display pricing table.
